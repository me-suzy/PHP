REQUIREMENTS:
-------------------------------
phpWebSite v0.9.0

-------------------------------
INSTALL:
-------------------------------
The comments module install is powered by the boost module installer.

1. Put comments into the mods directory of the phpWebSite base.
2. Point your browser at your site and go to the boost module.
3. Find Comment Manger in the list and click install.
4. Enjoy!

-------------------------------
UPGRADE:
-------------------------------
The comments upgrade is powered by the boost module.
(Not Implemented Yet)

1. Point your browser at your site and go to the boost module.
2. Find Comment Manager in the list and click check for upgrade.
3. Boost should do the rest.
4. Enjoy!
