REQUIREMENTS:
-------------------------------
phpWebSite v0.9.0

-------------------------------
INSTALL:
-------------------------------
The blockmaker module install is powered by the boost module installer.

1. Put blockmaker into the mods directory of the phpWebSite base.
2. Set you images directory in the conf/config.php file.
3. Point your browser at your site and go to the boost module.
4. Find blockmaker in the list and click install.
5. Enjoy!

-------------------------------
UPGRADE:
-------------------------------
The blockmaker upgrade is powered by the boost module.
(Not Implemented Yet)

1. Point your browser at your site and go to the boost module.
2. Find blockmaker in the list and click check for upgrade.
3. Boost should do the rest.
4. Enjoy!

-------------------------------
TEMPLATES
-------------------------------
There are no templates specifically for the blockmaker module.  Boxstyles
found in the themes are the templates for blocks.  Boxstyles can be set
within the layout module.
