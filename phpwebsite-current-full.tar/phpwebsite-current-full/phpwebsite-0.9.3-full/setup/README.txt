Installation instruction is contained in the docs/INSTALL.txt file.

Upgrade information in contained in the docs/UPGRADE.txt file.

phpWebSite is defaulted to run on a Unix operating system. If you
are installing on a Windows web server, make sure you read the portion
of the INSTALL.txt file that covers Pear pathing. If you do not alter
the referenced line, your installation will not run.

Help with installation problems may be answered by running repair.php
in this directory.

Please read the SECURE.txt file before attempting installation.

For extra security, we strongly recommend moving or restricting permissions
on this directory after installation.
