Album Settings

The settings for an album are found under the main album view. Next to the List Albums and New Photo link is a settings link. These settings are the same information you filled out when creating a new album. It is helpful to be able to get to this so you can possibly update the description of your album or hide it from the public while you add new photos to it. Just click the Save Album submit button to save any changes you may have made.
