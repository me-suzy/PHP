ShoutPro™ 1.0 BETA 1 - Installation Instructions

Installation Instructions:
  1. Open up the file "config.php" in a text-editing program and configure all of the settings to your liking. Each setting has a description of how it should be set and what it is.
  2. Open up your FTP client and create a folder on your web server for the shoutbox.
  3. Upload all of the files and folders found in the "upload" directory in this zip to the folder you just created.
  4. CHMOD the file "shouts.php" 666 (make it readable and writable by all). If problems arise CHMOD it 777 instead.
  5. Open the file "shoutbox.php" in your browser. Try posting a few messages and see if it works. If it doesn't repeat the past few steps.
  6. To put your new shoutbox on your site, just create an IFRAME with it. Here is a sample IFRAME code:

    <IFRAME frameborder="0" width="150" height="300" src="shoutpro/shoutbox.php"></IFRAME>

    Make sure the "src" attribute is pointing to the correct file and the width is what you specified in the configuration file.

  7. That's it! ShoutPro™ should now be working correctly. If you have any questions just send an e-mail to duffman@activegaming.net.

How to Do Some Simple Tasks with ShoutPro™:

  • Clear the Shoutbox: In your web browser, open the file "admin.php" in your ShoutPro™ folder. Enter the password you typed into "config.php" and hit submit. A message should appear saying the shoutbox has been cleared. The other features of admin.php are not complete in this beta version.
  • Add More Smilies: Begin by uploading the smilie image file into the "smilies" folder. Then open up "smilies.php" in the "lists" folder in a text-editing program. You can add or remove smilies here. Make sure you follow this format: "SMILIECODE|^|IMAGEFILE|^|". Each unit like that should be on a separate line.
  • Add/Edit ShoutCode and Profanity Filter: Open the file "lists/shoutcode.php" or "lists/profanities.php" in a text-editing program. These files are in the same format as the smiles.php file. In shoutcode.php, the first item is the ShoutCode tag and the second is the corresponding HTML tag. In profanities.php, the first item is the profanity and the second item is the filtered word.

Known Issues:

  • If a ShoutCode tag is left open in a shout, some problems may arise. This will be fixed in BETA 2.
